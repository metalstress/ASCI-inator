# ASCII-инатор

## Ключевые возможности
- Генерация ASCII-арта из изображений с поддержкой форматов JPG, PNG и JPEG, а также ручная настройка сетки, палитры из пяти цветов и фона, выбора шрифта и пользовательских наборов символов.
- Просмотр результата в окне предпросмотра с закруглёнными углами, поддержкой антиалиасинга и возможностью вывода на дополнительный монитор в полноэкранном режиме.

## Анимационные режимы
- Waves: классическая волновая анимация с управлением скоростью.
- Morph: плавное морфирование между двумя изображениями.
- Audioreactive: визуализация, реагирующая на звук с микрофона, включая выбор устройства ввода и настройку усиления.
- Contourswim: анимация плавающих контуров с параметрами чувствительности, размытия, скорости волн, амплитуды, количества слоёв и свечения.

## Постобработка и предпросмотр
- CRT Monitor: имитация старого монитора с настройками сканлайнов, виньетки, RGB-смещения и тряски.
- Glow: эффект свечения на основе двойного гауссова размытия.
- Accurate Preview: точный режим предпросмотра, отображающий итоговый результат экспорта.

## Экспорт
- Сохранение анимаций в GIF с настраиваемым FPS и скоростью проигрывания.
- Экспорт последовательностей PNG с качественным рендерингом через PIL.

## Пользовательский интерфейс
- Современный тёмный UI с прозрачным overlay, размытием и затемнением основного окна при открытии настроек.
- Настройки организованы в три вкладки (Интерфейс, Аудио, Производительность) с кастомными табами, плавными анимациями и отслеживанием изменений.
- Управление цветами интерфейса через HEX-поля и круглые пипетки, подбор шрифта, регулировка размера шрифта и аудио-параметров.

## Производительность
- Кэширование глифов для ускорения рендеринга и даунсемплинг предпросмотра.
- Пропуск кадров в режиме точного предпросмотра и GPU-ускоренные эффекты через QPainter.
- CPU-рендеринг высокого качества через PIL для итогового экспорта.

## Дополнительные возможности
- Генерация случайных палитр и импорт цветовых схем из изображений.
- Управление параметрами обнаружения контуров для режима Contourswim.
- Диалог загрузки с индикатором прогресса при экспорте и сохранение настроек в JSON.
